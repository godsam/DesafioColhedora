# DesafioColhedora
Criação de uma simulação de um processo de colhimento por meio de um trator utilizando unity e c#.

Dia 6
08/12/2017, dia que compreta as três semanas de prazo.

O que falta/ou:
- Interface. Não há interface para entrada de dados, nem como apontar local para gravação (não disponível).
- Salvar lista de dados coletados não ficou pronta (nem gera está ainda).
- Controle da camera.
- Animação/movimetação do "trator".
- Modelagem do trator.
- Comentários no código para facilitar compreenção do que queria fazer.

O que funciona:
- O sistema reconhece as areas de coleta.
- Ele pega os valores de acordo com a área definida no desafio.
- ele mostra no canto inferior esquerdo a area e o valor coletado, alem de mostrar os acumulados totais e por área (após 3 passagem de um ponto a outro, ele volta ao ponto inicial em um ciclo infinito, sem zerar os contadores).
